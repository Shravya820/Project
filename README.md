
# 🎬 AI Movie Assistant  

A smart movie recommendation web application that helps users discover the best movies and dramas from multiple industries such as Telugu, Tamil, Hindi, Korean, Hollywood, Japanese, Spanish, and more. Users can simply type their preferences and instantly receive curated movie suggestions with IMDb links, ratings, posters, reviews, and additional details.


---

# 1️⃣ Business Problem

Today users face difficulty while choosing movies because streaming platforms contain thousands of titles. It takes time to search, compare ratings, and decide what to watch.

### Main Challenges:

- Too many choices available  
- Hard to find movies based on mood  
- Need recommendations from different industries  
- Searching ratings and reviews separately  
- No quick personalized assistant  

---

# 2️⃣ Possible Solution

A smart assistant can simplify movie searching by allowing users to ask naturally:

- Best Telugu action movies  
- Korean romantic dramas  
- Funny Hindi comedy movies  
- Movies like Interstellar  
- Best thriller movies tonight  

The assistant can instantly recommend suitable movies with IMDb links and details.

---

# 3️⃣ Implemented Solution

We built **AI Movie Assistant**, a responsive chat-based website.

### Implemented Features:

✅ Chat-style interface  
✅ Movie suggestions by language  
✅ Genre based recommendations  
✅ Mood based recommendations  
✅ Korean movies & dramas  
✅ Telugu / Tamil / Hindi / Hollywood suggestions  
✅ IMDb links for posters, ratings, reviews  
✅ Typing animation  
✅ Mobile responsive UI  
✅ Optional n8n automation support

# 6️⃣ How to Run in Local

## Step 1: Clone Repository
```bash
git clone https://github.com/Shravya820/Project.git
```
## Step 2: Open Folder
```bash
cd movie_chatbot
```## Step 3: Install Packages
```bash
npm install
```
## Step 4: Run Project

```bash
npm run dev
```
## Step 5: Open Browser
```text
http://localhost:5173
```

# 🔟 Problems Faced and Solutions

| Slow response feeling      | Added typing indicator          |
| UI alignment issues        | Used Tailwind Flex/Grid         |

| Webhook errors             | Added local fallback logic      |
